ℹ️ About

About AEL Markdown CMS

A lightweight markdown editor by AEL Digital Studio

📋 Overview

AEL Markdown CMS is a client-side markdown editor with live preview, document management, and export. Built with a custom markdown parser — no external libraries.

⚡ Features

  • Live preview as you type
  • Split pane editor/preview
  • Toolbar for common markdown
  • Document save/load/delete
  • Export as HTML, MD, TXT
  • Auto-save to localStorage

🛠 Tech Stack

  • HTML5 — Semantic structure
  • CSS3 — Glassmorphism, split pane, responsive
  • JavaScript — Custom markdown parser, localStorage, File API
